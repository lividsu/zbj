# Image Generation Skill

## 适用场景
1. 用户纯文本要求生成新图片（文生图）。
2. 用户发送了参考图片，并要求进行普通的修改、编辑、风格化（如图生图）。
注意：如果是翻译语言请用 image_translate，如果是改分辨率请用 image_resize，如果是多图合成请用 image_composite。

## 处理方式
通过调用底层的图片生成/编辑模型，根据用户的提示词（可能结合参考图片）生成出新图片。
